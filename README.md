# CazadorRecon v20.4

**Kit OSINT para Termux** - Limpio, organizado y con TikTok OSINT

## Instalación

```bash
git clone https://github.com/CazadorRecon/CazadorRecon-Termux.git
cd CazadorRecon-Termux
pip install requests phonenumbers holehe
python cazadorrecon.py --menu
```

## Herramientas en el menú

- Sherlock
- Maigret
- Holehe
- WhatsApp OSINT
- Telegram OSINT
- **TikTok OSINT** (nuevo)
- Información de teléfono

## Otras herramientas recomendadas

### PhoneInfoga (Excelente para números)
```bash
git clone https://github.com/sundowndev/phoneinfoga.git
cd phoneinfoga && pip install -r requirements.txt
```

### theHarvester
```bash
pip install theHarvester
```

### Ignorant
```bash
pip install ignorant
```

### SocialScan
```bash
pip install socialscan
```

### ExifTool
```bash
pkg install exiftool
```

### Recon-ng
```bash
pkg install recon-ng
```
